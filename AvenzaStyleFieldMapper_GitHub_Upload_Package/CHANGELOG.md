# Changelog

## FieldMapperApp-14 MSTI / RVT Composite Update

### Added

- Added **Multi-Scale Topographic Index** as a terrain visualization option.
- Added **MSTI × VAT Composite** visualization option.
- Added MSTI max-scale parameter.
- Added MSTI multiply-strength parameter for combining MSTI with VAT.
- Added live-adjust support for MSTI composite settings.
- Added multiply-style terrain overlay support for grayscale RVT/VAT/MSTI products.
- Added transparency workflow for terrain visualizations over GeoPDFs, USGS topo maps, satellite/orthophoto imagery, and other basemaps.

### Intended use

Use MSTI to emphasize terrain texture and subtle multi-scale landform variation. Use MSTI × VAT Composite to combine the broad multi-scale terrain-index response with the local visualization strength of VAT.
