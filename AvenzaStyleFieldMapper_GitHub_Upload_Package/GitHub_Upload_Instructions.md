# GitHub upload instructions

## To upload this as a new GitHub repository

1. Go to GitHub and choose **New repository**.
2. Name it something like `AvenzaStyleFieldMapper`.
3. Create the repository.
4. Choose **uploading an existing file**.
5. Drag in:
   - `README.md`
   - `.gitignore`
   - `CHANGELOG.md`
   - the `Source` folder
   - the `docs` folder
6. Commit the files.

## To use this in your existing app repository

1. Open your existing GitHub repository.
2. Upload `Source/AvenzaStyleFieldMapperApp.swift`, or copy it over your current Swift app file.
3. Commit with a message like:

   `Add MSTI and RVT/VAT composite terrain visualization tools`

## To upload as a downloadable release

1. In GitHub, open your repository.
2. Go to **Releases**.
3. Choose **Draft a new release**.
4. Attach `AvenzaStyleFieldMapper_GitHub_Upload_Package.zip`.
5. Publish the release.
